# Git Hooks Quick Start

## Installation (Choose One)

### Option 1: Pre-Commit Framework (Recommended)
```bash
pip install --user pre-commit
bash scripts/install-hooks.sh
```

### Option 2: Husky (Node.js)
```bash
npm install
```

### Option 3: Both
```bash
bash scripts/install-hooks.sh --method=both
```

## Common Commands

```bash
# Test hooks without committing
pre-commit run --all-files

# Update hooks to latest versions
pre-commit autoupdate

# Test specific hook
pre-commit run golangci-lint --all-files

# Bypass hooks (emergency only!)
git commit --no-verify -m "message"
git push --no-verify

# Test commit message format
git commit --allow-empty -m "test: verify hooks"
```

## Commit Message Format

```
<type>(<scope>): <subject>

<body>

<footer>
```

**Valid types:** feat, fix, docs, style, refactor, perf, test, build, ci, chore, revert

### Examples

```
feat(wallet): add Fernet encryption support
fix(api): resolve connection timeout
docs: update README
```

## What Gets Checked

- ✅ Go: gofmt, goimports, go vet, golangci-lint, tests
- ✅ JavaScript: ESLint, Prettier
- ✅ Python: Black, Pylint, MyPy
- ✅ Trailing whitespace, file endings
- ✅ YAML/JSON syntax
- ✅ Sensitive data detection
- ✅ Commit message format

## Troubleshooting

```bash
# golangci-lint not found?
go install github.com/golangci/golangci-lint/cmd/golangci-lint@latest

# pre-commit not found?
pip install --user pre-commit
export PATH="$HOME/.local/bin:$PATH"

# Hooks running slow?
SKIP=golangci-lint git commit -m "message"
```
