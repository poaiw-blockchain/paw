Getting Started

Base URLs

  • Mainnet: https://api.paw.network
  • Testnet: https://testnet-api.paw.network
  • Local: http://localhost:1317

Rate Limits

Public endpoints: 100 requests/minute

Authenticated endpoints: 1000 requests/minute

Quick Start Example

# Get account balance
curl http://localhost:1317/cosmos/bank/v1beta1/balances/paw1abc123...

# List all DEX pools
curl http://localhost:1317/paw/dex/v1/pools

# Get Oracle price feed
curl http://localhost:1317/paw/oracle/v1/prices/BTC/USD
// Using fetch API
const baseURL = 'http://localhost:1317';

// Get account balance
const balance = await fetch(
  `${baseURL}/cosmos/bank/v1beta1/balances/paw1abc123...`
).then(r => r.json());

// List DEX pools
const pools = await fetch(
  `${baseURL}/paw/dex/v1/pools`
).then(r => r.json());

console.log('Pools:', pools);
import requests

base_url = 'http://localhost:1317'

# Get account balance
response = requests.get(
    f'{base_url}/cosmos/bank/v1beta1/balances/paw1abc123...'
)
balance = response.json()

# List DEX pools
response = requests.get(f'{base_url}/paw/dex/v1/pools')
pools = response.json()

print('Pools:', pools)
package main

import (
    "encoding/json"
    "net/http"
)

const baseURL = "http://localhost:1317"

func main() {
    // Get account balance
    resp, _ := http.Get(baseURL + "/cosmos/bank/v1beta1/balances/paw1abc123...")
    var balance map[string]interface{}
    json.NewDecoder(resp.Body).Decode(&balance)

    // List DEX pools
    resp, _ = http.Get(baseURL + "/paw/dex/v1/pools")
    var pools map[string]interface{}
    json.NewDecoder(resp.Body).Decode(&pools)
}

API Modules

DEX Module

Native decentralized exchange with AMM pools and atomic swaps

  • GET /paw/dex/v1/pools
  • GET /paw/dex/v1/pools/{id}
  • POST /paw/dex/v1/create_pool
  • POST /paw/dex/v1/swap
  • POST /paw/dex/v1/add_liquidity

Oracle Module

Decentralized price feeds with median aggregation

  • GET /paw/oracle/v1/prices
  • GET /paw/oracle/v1/prices/{asset}
  • POST /paw/oracle/v1/submit_price
  • GET /paw/oracle/v1/params

Compute Module

TEE-protected compute aggregation and task management

  • GET /paw/compute/v1/tasks
  • GET /paw/compute/v1/tasks/{id}
  • POST /paw/compute/v1/submit_task
  • GET /paw/compute/v1/providers

Cosmos SDK

Standard Cosmos SDK modules (bank, staking, governance)

  • GET /cosmos/bank/v1beta1/balances/{address}
  • GET /cosmos/staking/v1beta1/validators
  • GET /cosmos/gov/v1beta1/proposals
  • GET /cosmos/auth/v1beta1/accounts/{address}
