# Devnet State Directory

This directory is auto-generated by `init_node.sh` and contains:

- `genesis.json` - Network genesis file
- `node*.id` - Node identifiers
- `node*_validator.mnemonic` - Validator mnemonics (SENSITIVE)
- `faucet.mnemonic` - Faucet wallet mnemonic (SENSITIVE)
- `*_key.yaml` - Key export files (SENSITIVE)
- `*.priv_validator_key.json` - Validator private keys (SENSITIVE)

## Security Warning

**Never commit actual `.state/` contents to git!**

The `.state/` directory is excluded in `.gitignore` because it contains sensitive cryptographic material.

## Generating New State

Run the devnet setup script:

```bash
./scripts/devnet/init_node.sh
```

This will create fresh keys and genesis for local testing.
